## Image Guidelines

- **Format**: PNG for screenshots, GIF for animations
- **Size**: Optimize images (use tools like TinyPNG)
- **Naming**: Use descriptive lowercase names with hyphens
- **Resolution**: Keep original device resolution when possible
- **Annotations**: Add arrows or text to highlight features (optional)

## How to Capture

1. On your Garmin watch: Take screenshot (if supported)
2. Via Garmin Connect: Export from watch face preview
3. Via Simulator: Capture from Connect IQ SDK simulator
