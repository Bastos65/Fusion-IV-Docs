# Changelog - FUSION IV

All notable changes to FUSION IV watch face will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-02

### Initial Stable Release

### Added
- Initial beta release
- Support for 38 Garmin watch models
- 73+ configurable data types
- 28 independently customizable colors
- Dual weather provider support (Garmin Weather + OpenWeatherMap)
- 18 configurable data field zones
  - 4 circular fields (9h, 11h, 1h, 3h)
  - 6 classic fields (3 top, 3 bottom)
  - 2 text fields (top, bottom)
  - 5 information indicators
  - 1 goal progress bar
- Multiple display modes for each field
- Time display customization
  - 12h/24h format
  - 3 seconds display modes
  - Leading zero option
  - Independent color for each time component
- Date display with 7 format options
- 19 language translations
- Power saving features
  - Screen saver delay
  - Screen off delay
  - 4 refresh rate options
- Font size customization (5 independent ratios)
- Custom text fields support
- Countdown timer
- 4 additional time zones
- Moon phase and sun events tracking
- Icon sets (filled/outlined)
- Battery adaptive colors
- Weather colored icons
- Touch screen widget launcher

### Features
- **Activity Tracking**
  - Steps, distance, calories
  - Active minutes (daily/weekly)
  - Floors climbed
  - Sport-specific distances (bike, run)
  
- **Health Monitoring**
  - Heart rate with min/max tracking
  - Body battery
  - Stress level
  - Respiration rate
  - Oxygen saturation (SpO2)
  - Recovery time
  
- **Performance Metrics**
  - VO2 Max (running/cycling)
  - Race predictors (5K, 10K, half, full marathon)
  - Training status
  
- **Environmental Data**
  - Altitude, pressure, temperature
  - Comprehensive weather information
  - Sun/moon events
  
- **System Information**
  - Battery level (days/percentage)
  - Solar intensity
  - Connectivity status
  - Notifications and alarms count
  - Do Not Disturb status

### Technical
- Minimum API Level: 5.0.0
- Language: Monkey C
- Framework: Garmin Connect IQ SDK
- Permissions: Background, Communications, ComplicationSubscriber, Positioning, SensorHistory, UserProfile
